Legend

  • Clockwise -> Incoming call
  • Counter clockwise -> Outgoing call

  • Size -> Length of the call

  • Green -> Family
  • Blue -> Friends
  • Purple -> Other

×

This project visualizes cellular data in an interactive way. The orbit represents different calls, with the direction of the orbit indicating whether the call was incoming or outgoing. The size of the orbit represents the length of the call. The color of the orbit indicates the category of the call: green for family, blue for friends, and purple for others. You can filter the data by duration using the slider, or by category using the checkboxes.
